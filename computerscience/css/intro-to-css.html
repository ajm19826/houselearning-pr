Introduction to CSS

What is CSS?

CSS (Cascading Style Sheets) is a language used to describe the style of an HTML document. It controls layout, colors, fonts, spacing, and more.

1. How to Apply CSS

There are three ways to apply CSS:

  • Inline CSS: In the HTML tag‚Äôs style attribute.
  • Internal CSS: Inside the <style> tag in the HTML head.
  • External CSS: In a separate .css file linked to the HTML.

2. Basic CSS Syntax

The CSS syntax follows this pattern:

selector {
  property: value;
}

Example:

p {
  color: red;
}

This sets all <p> text color to red.

3. CSS Selectors

Selectors target HTML elements:

  • Element selector: Targets tags like p, h1.
  • Class selector: Targets elements with a class, e.g. .myClass.
  • ID selector: Targets elements with an ID, e.g. #myId.

4. Styling Text and Boxes

Common properties:

  • color: Text color.
  • background-color: Background color.
  • font-size: Text size.
  • border: Border styling.
  • padding: Space inside element.
  • margin: Space outside element.

6. CSS Box Model

The box model parts:

  • Content: The stuff inside (text, images).
  • Padding: Space between content and border.
  • Border: The border itself.
  • Margin: Space outside the border.
