What is the CSS Box Model?

The CSS Box Model wraps around every HTML element. It‚Äôs made of:

  • Content: The actual content (text, images, etc.)
  • Padding: Space around the content
  • Border: A border surrounding the padding
  • Margin: Space around the border

Box Model Example

This box uses padding, border, and margin.

  • Width: 200px
  • Height: 200px
  • Padding: 20px
  • Border: 10px solid
  • Margin: 20px
üí° Tip: By default, width/height only cover the content area unless you use box-sizing: border-box;.

Understanding box-sizing

The default value is content-box, which means width and height only apply to the content area.

If you use box-sizing: border-box;, padding and border are included in the total size.

.myBox {
  width: 200px;
  padding: 20px;
  border: 10px solid #333;
  box-sizing: border-box;
}
