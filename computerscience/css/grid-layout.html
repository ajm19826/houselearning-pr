CSS Grid Layout

What is CSS Grid Layout?

CSS Grid Layout is a two-dimensional layout system for the web, meaning it can handle both columns and rows. It's used to create complex and responsive web designs with ease. The grid layout allows you to place elements into rows and columns, controlling their size, placement, and spacing.

Grid Container Properties

  • display: grid; - Defines an element as a grid container.
  • grid-template-columns: Defines the number and size of the columns.
  • grid-template-rows: Defines the number and size of the rows.
  • grid-gap: Defines the space between the grid items.
  • grid-template-areas: Defines named grid areas.
