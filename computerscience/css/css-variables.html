How CSS Variables Work

:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --font-size: 18px;
  --box-size: 150px;
}

.box {
  background-color: var(--secondary-color);
  font-size: var(--font-size);
  width: var(--box-size);
  height: var(--box-size);
}

Responsive Variables Example

@media (max-width: 600px) {
  :root {
    --box-size: 100px;
    --font-size: 14px;
  }
}
