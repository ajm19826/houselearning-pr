Lesson: JavaScript Functions

What are Functions in JavaScript?

Functions are one of the core building blocks of JavaScript. They allow you to group code into reusable chunks. A function can take input (parameters), perform actions, and return output.

Functions help you avoid code repetition, and can make your code more readable and maintainable.

Here's a simple function example: 

function greet(name) {
    return "Hello, " + name + "!";
}

let greetingMessage = greet("Alice");
console.log(greetingMessage);  // Output: "Hello, Alice!"
