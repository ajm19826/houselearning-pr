JavaScript Variables & Data Types

Variables store data in JavaScript. You can declare variables using var, let, or const.

JavaScript has different data types, such as:

  • String: "Hello World"
  • Number: 42
  • Boolean: true or false
  • Array: [1, 2, 3]
  • Object: { name: "Alice", age: 25 }
