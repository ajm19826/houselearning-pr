Understanding Meta Tags and SEO

Meta tags are snippets of text that describe a web page's content. They're placed in the <head> section of your HTML. While meta tags don't directly affect how a page looks, they give search engines info that improves SEO (Search Engine Optimization).

  • <meta charset="UTF-8"> - Specifies character encoding (super important for languages and emojis).
  • <meta name="description" content="..."> - Short page summary used in search results.
  • <meta name="keywords" content="..."> - Keywords for search engines (less relevant nowadays but still used sometimes).
  • <meta name="author" content="..."> - Who made this page.
  • <meta name="robots" content="index, follow"> - Controls if search engines index your page and follow links.
  • <meta property="og:title" content="..."/> - Open Graph tag for social media previews (Facebook, Twitter, etc.).
