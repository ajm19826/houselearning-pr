What's New in HTML5?

HTML5 introduced a variety of new features that make web development more powerful, easier, and more efficient.

  • <header> - Represents the header of a document or section.
  • <footer> - Defines the footer for a document or section.
  • <article> - Represents a self-contained piece of content.
  • <section> - Represents a thematic grouping of content.
  • <nav> - Defines a navigation bar for links.
  • <video> - Embeds video content directly into a web page.
  • <audio> - Embeds audio content directly into a web page.
  • <canvas> - Provides an area to draw graphics dynamically using JavaScript.
  • <figure> - Represents self-contained content, such as an image or chart.
  • <mark> - Highlights text that has been marked or emphasized.
  • <progress> - Represents the progress of a task or action.
