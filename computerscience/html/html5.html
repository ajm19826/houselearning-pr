🚀 HTML5 New Features

📚 What’s New in HTML5?

HTML5 added new elements that make structuring, multimedia handling, and dynamic content easier.

  • <header> — Defines a header for a page or section
  • <footer> — Defines a footer for a page or section
  • <article> — Self-contained content block
  • <section> — Thematic grouping of content
  • <nav> — Navigation links section
  • <video> — Embed video content
  • <audio> — Embed audio content
  • <canvas> — Draw graphics with JS
  • <figure> — Image/diagram with optional caption
  • <mark> — Highlighted text
  • <progress> — Progress bar for tasks
