Understanding Ordered and Unordered Lists

HTML provides two types of lists:

  • <ul> - An unordered list (bullets).
  • <ol> - An ordered list (numbers).
  • <li> - List item (used in both ordered and unordered lists).

Ordered List Example

        <ol>
            <li>First item</li>
            <li>Second item</li>
            <li>Third item</li>
        </ol>

Unordered List Example

        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ul>
