📚 How Images & Media Work in HTML

The <img> tag is used to display images, while videos and audio are embedded using their own tags.

  • <img> — Embeds an image
  • src — Specifies image source (URL or file path)
  • alt — Fallback text if image fails to load
  • width / height — Image sizing attributes
