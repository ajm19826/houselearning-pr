🔧 Understanding HTML Elements

HTML elements are the building blocks of web pages. Each element has:

  • A start tag
  • Some content
  • An end tag

Common HTML Elements:

  • <p> — Defines a paragraph
  • <h1> to <h6> — Headings
  • <a> — Hyperlink
  • <img> — Image
  • <ul> and <li> — Lists

📑 HTML Document Structure

<!DOCTYPE html>
<html>
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph in HTML.</p>
  </body>
</html>
