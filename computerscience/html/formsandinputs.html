🎉 HTML Forms & Inputs

📋 What Are Forms?

HTML forms collect user input through different form controls like text fields, radio buttons, checkboxes, and buttons.

  • <form> — Defines an HTML form
  • <input type="text"> — Text input field
  • <input type="radio"> — Radio button
  • <input type="checkbox"> — Checkbox
  • <button> — Clickable button
