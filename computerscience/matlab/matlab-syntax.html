üìè MATLAB Syntax & Commands

MATLAB scripts are a series of commands saved in a file (with a .m extension). You can also run commands interactively in the Command Window.

üìñ Key Syntax Rules

  • Statements do not require a semicolon, but ending a line with ; suppresses the output.
  • Use % to start a comment.
  • Variable names are case-sensitive.
  • Functions are saved in separate .m files, named after the function.
